## Shape mechanics

### The five open forms

The CAGED system is named after the five open-position chord forms that cover the entire fretboard when used as movable barre chords:

| Form | Open position | Root location |
|------|--------------|---------------|
| C | Open C chord | 5th string, 3rd fret |
| A | Open A chord | 5th string, open |
| G | Open G chord | 6th string, 3rd fret |
| E | Open E chord | 6th string, open |
| D | Open D chord | 4th string, open |

The "X" strings (strings that should not be strummed) vary by form: C form mutes 6th string; D form mutes 5th and 6th strings; A and E forms use all strings; G form uses all strings.

**Root is always the lowest pitched note** in the form (the note where you'd start strumming). Knowing which string carries the root is the key to transposing each shape to any key.

### How the shapes connect (the CAGED sequence)

Moving up the neck from any starting point, the five shapes appear in the sequence: C → A → G → E → D → (C again, one octave higher). Each shape's upper boundary overlaps with the next shape's lower boundary — the shapes interlock.

- The C form's root is on the 5th string. Move up from C-form to the A-form: the A-form root is also on the 5th string but higher up.
- The G-form root moves to the 6th string.
- The E-form root is on the 6th string.
- The D-form root is on the 4th string (and also the 2nd string).
- D-form connects back into C-form an octave higher.

**Practical implication:** If you know where any one CAGED shape is for a given note, you can locate all five shapes for that note in sequence across the neck.

### Barre chord transformation

Each open form becomes a movable barre chord by replacing the nut (open position) with the index finger as a capo. The index finger only needs to hold down the strings that were open in the original form — it doesn't have to fully barre all six strings.

- C form: index finger holds strings 1 and 3 (the nut strings in open C). Pinky on 5th string is the root. Rarely played in full — usually used as a partial form.
- A form: full index-finger barre is common; all other fingers build the A shape on top.
- G form: index finger holds strings 1 and 6 (or all six for a full barre). Root is on 6th string.
- E form: most common barre chord — full index barre, A-shape or A-minor-shape built with remaining fingers. Root is on 6th string.
- D form: index finger on strings 1–4; root is on 4th string.

### Partial forms and inversions

Breaking a full CAGED form into a partial form creates:
- Different inversions (different note in bass)
- Easier-to-play voicings
- Unique shapes and sounds used extensively in popular music

**C-form partial examples (Desi Serna):**
- Remove pinky → 4-note voicing on strings 1–4 ("Stairway to Heaven" intro)
- Remove pinky + ring → 3-note voicing on strings 1–3 (same triangular shape as partial D-form; "Hole Hearted" by Extreme)
- C-form and D-form share this triangular partial shape

**Keith Richards trick:** Play partial C-form without index and pinky on strings one and five respectively — leaves a 3-note voicing on strings 2, 3, 4. Enables easy alternation between C-form and A-form with minimal hand movement.

**Alternate bass notes:** Adding the open 6th string to a C chord creates C/E. Adding a G on the 6th string creates C/G. These slash chords are common voice-leading devices.

### Visualizing the root when it's omitted

When playing a partial form that omits the root, always visualize where the root would be in the parent form. This root awareness is what ties the partial shape to its chord name and allows transposition.

## Chord-scale relationships per shape

### Pentatonic patterns anchored to CAGED shapes (Desi Serna)

The five pentatonic patterns align with the five CAGED shapes. Each CAGED shape "lives" in the same fretboard zone as one of the five pentatonic patterns. This alignment is why the CAGED system and pentatonic patterns reinforce each other — learning both systems simultaneously is more efficient than learning them separately.

**Key connection:** Pentatonic pattern 1 aligns with the E-form CAGED shape. The root of pattern 1 (minor root) corresponds to the E-form root on string 6. The major root in pattern 1 corresponds to the D-form or G-form root in that zone.

### Scale extensions from CAGED shapes (Leavitt)

The four major scale fingering types (Types 1–4) correspond to CAGED-zone positions:
- Each fingering type is playable in a specific CAGED shape's home zone.
- Converting a major scale type to Dorian, Phrygian, or other modes requires modifying specific notes within the same hand position.
- This means: **the CAGED system provides the positional anchor, and the scale type provides the note choice within that anchor.**

### Drop-2 voicings per chord scale (Voice Leading — John Thomas)

Drop-2 voicings are derived from CAGED-zone chord shapes. They are built by taking a close-position 4-note chord and dropping the second highest voice one octave. This creates guitar-friendly voicings across strings 2–5 or 1–4.

For each chord function (Ionian, Dorian, Mixolydian, etc.), the drop-2 voicings produce a set of movable grips that can be practiced chromatically up and down the neck — this is a direct application of the CAGED transposition principle.

**Chord families from Voice Leading:**
1. Major chord family: Major → Maj7 → Dom7 → Maj6 → Min(b6) in a single voice-leading stream
2. Minor chord family: Minor → Min(Maj7) → Min7 → Min6 → Min(b6)
3. Dominant 7 family: Dom7 → Dom7(13) → Dom7(#11) → etc.
4. Altered dominant family: builds from same root with altered tensions

## Teaching order and common progressions through shapes

### Open chord sequence (Leavitt, Fretboard Theory)

The natural learning progression for open-position chords:
1. **E and A forms** — most common barre chord forms; root on 6th and 5th string respectively. Start here.
2. **D form** — introduces root on 4th string; connects to C and A forms.
3. **C form** — the most challenging full barre; usually learned as a partial form first.
4. **G form** — introduces the full 6-string barre with root on 6th string at a different shape.

Leavitt's derivative chord sequence (Vol. 1): New chord forms are presented in a sequence where each new form is directly related to one or more preceding forms (by moving or removing specific fingers). This "derivative" approach means: **memorize the fingering relationship, not isolated shapes**.

### The CAGED sequence as a practice tool

To learn all positions of a chord type:
1. Find the note on string 6 (E-form root)
2. Play E-form barre chord
3. Move up to G-form (root shifts to string 6 higher up) — or D-form (root to string 4)
4. Continue through C-form (root to string 5), then A-form (root string 5)
5. Arrive at next E-form one octave higher

Cycling through this sequence in one key is a standard fretboard coverage exercise.

### Common chord progressions in CAGED context

**I–IV–V (major):** In any given CAGED zone, play the I-chord shape, then locate the IV and V chords in nearby shapes. The IV chord is always in the CAGED shape one step "earlier" in the sequence (e.g., if I is E-form, IV is often a D-form higher up). The V chord uses the shape one step "later" or lower.

**I–VI–II–V (jazz turnaround):** Natural CAGED motion with voice-led inner voices. Each chord's CAGED shape should be chosen to minimize hand movement — select shapes that share common tones.

**Blues I–IV–V:** Most guitar blues relies on E-form or A-form barre chords for I and IV, with V-form (A or E) nearby. The blues shuffle uses partial forms (low E and A strings only).

### Chord-shape-to-scale teaching order (for SlopScale pathways)

Recommended introduction order for SlopScale CAGED-based pathways:
1. **E-shape** — most common movable shape; pentatonic pattern 1 aligns here
2. **A-shape** — second most common; pentatonic pattern 2 aligns here
3. **D-shape** — introduces 4th-string root; pentatonic pattern 3 zone
4. **C-shape** — harder barre; pentatonic pattern 4 zone
5. **G-shape** — large stretch; pentatonic pattern 5 zone

This mirrors the SlopScale current CAGED_SHAPES implementation and is validated by both Leavitt (E and A forms first for barre chord practice) and Fretboard Theory (E-form as default starting CAGED shape).

## Practice implications for SlopScale

1. **CAGED shapes are positional anchors, not just chord voicings.** Each shape defines a zone of the fretboard within which scales, arpeggios, and chord voicings all live. SlopScale exercises set at a specific CAGED position should present all relevant material (scale + arpeggio + chord voicing) within that zone.

2. **Root-string awareness is critical.** For SlopScale's `CAGED_SHAPES` data, the `rootStringIdx` field is the key parameter — it determines where to place the shape for a given key. Always verify which string carries the root before placing a shape.

3. **Partial forms are the most practical guitar reality.** Full barre chords (especially C-form) are often not how experienced players voice chords. SlopScale chord-tone exercises could favor partial forms and inversions over always requiring the full shape.

4. **Derivative chord learning reduces memorization burden.** Teaching chord forms as modifications of a base form (add a finger, move a finger) rather than as isolated new shapes aligns with Leavitt and reduces total memorization load. This philosophy should inform how SlopScale presents "Next Shape" in a pathway.

5. **Five-shape coverage should be the long-term goal of any pathway.** No pathway is complete until the student can play the same material in all five CAGED zones. Pathways with `vary[]` should cycle through shapes systematically, not randomly.

6. **Drop-2 voicings are the practical jazz guitar application of CAGED.** If SlopScale ever adds a chord-voicing exercise type, the drop-2 approach (play the same chord quality through all five CAGED shapes using voice-led drop-2 voicings) is the most established and musically useful format.
